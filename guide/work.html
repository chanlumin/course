# 组件化编程实战

Vue.js提供组件注册功能，描述组件的功能, 那么什么是Vue.js ？Vue.js是一个前端框架，它是一个注重View层的MVVM框架。前端界面的数据与JavaScript模型对象的数据都是通过ViewModel来进行同步更新，ViewModel就是一个中介视图与模型数据的绑定，当数据在两边改变时，不需要我们手动获取DOM或者模型进行更新，VM自动同步两边的数据，不用手动获取DOM元素再重新赋值更新

一个最简单的Hi Veena的数据绑定Vue.js组件如下：

<template>
  <div class=”wrapper”>
    {{message}}
  </div>
</template>
<script>
export default {
  data(){
    return {
      message: “Hi Veena”
    }
  }
  components:{}
}

</script>
<style>.wrapper {height:100%;width:100%;background:#fefefd}</style>

template里面的内容为HTML元素，HTML中{{}}里面的字符串是变量，这个变量和data里面返回的属性message绑定在一起，Vue组件完全渲染在HTML时，message也会跟着被渲染进去。 使用Vue组件的方式执行 import Veena from ‘Veena.vue’在components:{Veena}在HTML中就可以用显示 Hello Veena的标签组件。 了解基本的组件之后，开始项目实战案例。安装好node.js，NPM和git然后从https://github.com/chanlumin/course_example.git 下载写好的初始化项目。执行git clone https://github.com/chanlumin/course_example.git 然后进入course_example项目执行npm install安装项目依赖。 提供的组件都在src/components下，需要实现的网站在src/views/Home.vue 在Home.vue的script标签里面添加以下代码引入对应的模块

import {Col,Row} from ‘@/components/layout’
import UHeader from ‘@/components/UHeader
import Swiper from ‘@/components/Swiper
import UContent from ‘@/componets/UContent
import UCard from ‘@/components/UCard’
import Toast from ‘@/component/Toast’
import UFooter from ‘@/components/UFooter’

在components:{ UHeader , Swiper , UContent , UCard, Toast , UFooter, Row,Col } 进行注册登记就可以在template里面使用组件。 由于导航UHeader属于全局的通用组件，所以导航组件应在app.vue当中进行引入即可引入如图5-3所示，先import引入UHeader接着在app的下面以的形式引入即可
