# 编程介绍

编程就是和计算机进行对话，达到和计算机沟通的目的。这个对话的过程，要使用某种计算机听得懂的语言去进行，并且细节要讲清楚。从科学的角度来说，编程是一种逻辑思维的运算，因为每一种语言都有一种固定的逻辑。生活中随处可见的手机APP、网站应用、机器人、红绿灯调度系统等都是通过编程实现的，本章要介绍的网站应用也是通过编程实现。 21世界是计算机发生巨大变革的时代，编程让这个世界变得更小了，更方便了。微信让运营商不再垄断通信通话；小米物联网智能生态链让生活更方便：智能家居、智能空调、小爱同学、外出公交卡，平衡车极大改善和便利日常生活；搜索让检索获取更加容易；网上知识问答社区让知识获取变得亲民；各种娱乐软件的出现极大丰富了老百姓的精神需求。编程已经融入生活了，在这个时代了解编程可以拉近你与科技和生活的距离。

为什要学编程呢？编程能够培养学生的创造能力、逻辑思维能力和解决问题的能力。在编写程序逻辑的过程中，是从无到有实现创建一个新的事物，当完成一个作品时候可以获得极大的成就感以及满足感。在编程的过程中，经常会遇到很多的问题，出了问题需要自己上网进行搜索，阅读理解和消化相关的知识，然后再回去解决相关问题，在这个过程中培养了孩子自己解决问题的能力，对以后孩子的独立性成长很有帮助。编程是一件有趣而又有挑战的事情，即使你未来不需要从事编程这方面的工作，通过学习编程培养的逻辑思维、创造能力和解决问题能力都会对你将来的事业有所帮助
